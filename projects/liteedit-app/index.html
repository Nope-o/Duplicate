LiteEdit Complete Guide

Complete walkthrough for import, editing, text tools, and export workflows.

Start & Import
  • Import from Uploads panel, Add Media (+), drag-and-drop, or camera capture.
  • You can work with up to 20 images in one session.
  • Use the left Uploads side tab if uploads panel is hidden.
Modes & Panels
  • Simple mode is for quick edits; Advanced mode unlocks full controls.
  • Use Hide in Editor Controls header to close the panel.
  • Use the right Edit Controls side tab to reopen it.
Image Editing
  • Use top toolbar for Pen, Highlighter, Blur, and Crop.
  • Crop mode has ratio, apply, and cancel actions.
  • Use undo/redo whenever available for safe iteration.
Smart Text Edit
  • Tap Detect Text, then select blocks directly on image.
  • Edit from the on-image editor for faster workflow.
  • Keep Size preserves style/scale; drag selected block to reposition.
  • Long replacement text can extend box width when needed.
Export
  • Use top-right Export for Single or Bulk output.
  • Adjust format, quality, metadata strip, and PDF mode.
  • Check size estimate and progress before final save.
Performance Tips
  • Keep import optimization enabled for large photos.
  • Use lower max edge on low-memory mobile devices.
  • Hide extra panels on smaller screens for smoother editing.
